# Code of Conduct — Repo‑Map Repository

This repository is part of the SCU ecosystem and follows the SCU Foundation’s global Code of Conduct.

All contributors MUST:

- act respectfully and professionally  
- avoid harassment, discrimination, and abusive behaviour  
- collaborate in good faith  
- respect governance decisions and update sequences  
- follow the SCU Foundation’s safety and trust principles  

For governance and escalation, this repository defers to:

- REPO: Governance — /governance-charter.md  
- REPO: Governance — /change-management-charter.md  
- REPO: Governance — /safety-governance.md  

Violations may result in restricted access or removal of contributions.
